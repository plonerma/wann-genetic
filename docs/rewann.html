rewann package¶

class rewann.Environment(params)[source]¶

Environment for executing training and post training evaluations.

Takes care of process pool, reporting, and experiment parameters.

class rewann.Individual(genes=None, network=None, measurements=None, id=None, birth=None, parent=None, mutations=0)[source]¶

Collection of representations of an individual.

Will contain genes (genotype), network (phenotype), and performance statistics (fitness).

genes : rewann.Genes network : rewann.Network, optional id : int, optional birth: int, optional

Index of the generation the individual was created

parentint, optional

Id of the parent individual (self is result of a mutation on parent)

mutationsint, optional

Length of the chain of mutations that led to this individual

measurementsdict

Measurements that have already been made on the individual (might be overwritten)
