Copy Task¶

Copy task as described in [ASB15].

The alphabet consists of 10 symbols ( \(\Sigma = \{ 0,...,7, e, d \}\)). Eight of those represent the available categories (\(\{0,..,7\}\)) that are supposed to be reproduced. \(e\) symbolizes an empty space, and \(d\) is the delimiter. Each sequence is (\(T+20\)) elements long: 10 elements for the input, \(T\) elements memorization phase (the last element in that phase is \(d\)), and 10 empty elements during which the network will need to reproduce the input sequence.

E.g.:

input:           6976018327eee ... eeedeeeeeeeeee
expected output: eeeeeeeeeeeee ... eeee6976018327
                 |  (in)   | (waiting) |  (out)  |
